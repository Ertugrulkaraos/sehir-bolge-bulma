# Şehir Bölge Bulma Uygulaması

Bu proje, girilen şehrin Türkiye'nin hangi bölgesinde olduğunu gösteren basit bir C# konsol uygulamasıdır.

## Özellikler
- Kullanıcıdan şehir adı alır.
- Şehri küçük harfe çevirir.
- Şehrin hangi bölgede olduğunu belirler.
- Bulunamazsa uyarı verir.

## Çalıştırma
Visual Studio veya `dotnet` CLI ile çalıştırabilirsiniz.

```
dotnet run
```
